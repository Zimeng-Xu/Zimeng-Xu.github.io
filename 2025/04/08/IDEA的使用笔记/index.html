IDEA常用快捷键

1.删除当前行：Ctrl+Y

2.复制当前行（自己设定）：Ctrl+Alt+空格，空格

3.补全代码：Alt+/

4.添加注释和取消注释（第一次添加，第二次取消）：Ctrl+/

5.自动导入该行需要的类：先配置auto import，然后使用Alt+Enter即可

6.快速格式化代码：Ctrl+Alt+L

7.快速运行程序（自己设定）：Alt+R

8.生成构造器等：Alt+Insert

9.查看一个类的层级关系：Ctrl+H

10.将光标放在一个方法上，输入Ctrl+B，可以定位到方法。

11.自动分配变量名，通过在后面加.var

模板/自定义模板

模板查看路径：

file -> settings -> editor -> Live templates
