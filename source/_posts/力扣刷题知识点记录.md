2. >给你一个整数 x ，如果 x 是一个回文整数，返回 true ；否则，返回 false 。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

  ```java
  class Solution {
    public boolean isPalindrome(int x) {
        if (x < 0 || (x % 10 == 0 && x != 0)) {
            return false;
        }

        int temp = 0;
        while (temp < x) {
            temp = temp * 10 + x % 10;
            x /= 10;
        }

        return temp == x || temp / 10 == x;
    }
  }
  ```
  
  中间的while循环实现数字的反转。
